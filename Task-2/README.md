#  LangChain Basics Demo

This project demonstrates core LangChain concepts:

- LLM Calls
- Prompt Templates
- Chains
- Memory
- Simple Agent (concept)

---

## Features
LLM Call → Basic model interaction (API required)
PromptTemplate → Dynamic prompts
Chain → Simple workflow simulation
Memory → Store past interactions
Agent → Conceptual example

---

## Conclusion

A simple starter project to understand LangChain fundamentals.
