#  Prompt Engineering Playground using LangChain

A comprehensive hands-on project that demonstrates how to build a **dynamic prompt generation system** using LangChain. This project covers everything from basic prompt templates to a fully functional prompt pipeline with validation and chat-based roles.

---

##  Project Objective

The goal of this project is to understand and implement:

- Prompt engineering fundamentals  
- Dynamic prompt generation  
- Multi-input prompt systems  
- Prompt variations (teaching, storytelling, interview)  
- Chat-based prompting  
- Input validation  
- End-to-end prompt pipeline
  
  ---
  ##  Tech Stack

- **Python 3**
- **LangChain**
- **Google Colab**
